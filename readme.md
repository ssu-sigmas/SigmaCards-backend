# SigmaCards Backend 🧠✨

Добро пожаловать в бэкенд-репозиторий **SigmaCards** — сервиса для умного запоминания с использованием интервальных повторений.

Этот проект предоставляет мощный API для управления карточками, колодами и процессом обучения, используя передовой алгоритм **FSRS (Free Spaced Repetition Scheduler)**.

---

## 🚀 Особенности

- **🔐 JWT Авторизация**: Безопасная система аутентификации с использованием Access и Refresh токенов.
- **🧠 Умный алгоритм FSRS**: Использование современного алгоритма интервальных повторений (лучше, чем стандартный SM-2 в Anki) для максимальной эффективности запоминания.
- **⚡️ Высокая производительность**: Написано на Python с использованием асинхронности.
- **🐳 Docker Ready**: Полная контейнеризация для быстрого развертывания.
- **📊 Статистика**: Отслеживание прогресса обучения и эффективности запоминания.

---

## 🛠 Стек технологий

- **Язык**: Python 3.11+
- **База данных**: PostgreSQL
- **ORM**: SQLAlchemy + Alembic (миграции)
- **Контейнеризация**: Docker & Docker Compose
- **Алгоритм**: FSRS (Free Spaced Repetition Scheduler)

## 🏁 Быстрый старт

### Предварительные требования

- Docker и Docker Compose
- Git

### Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ssu-sigmas/SigmaCards-backend.git
   cd sigmacards-backend
   ```

2. **Настройте переменные окружения:**
   Создайте файл `.env` на основе примера:
   ```bash
   cp .env.example .env
   ```
   Заполните `.env` своими данными (БД, секретные ключи и т.д.).

3. **Запустите проект:**
   ```bash
   docker-compose up --build
   ```

4. **Примените миграции:**
   ```bash
   docker-compose exec app alembic upgrade head
   ```

Сервис будет доступен по адресу: `http://localhost:8000` (или порт, который вы указали).

## 🔑 Авторизация (JWT)

Сервис использует пару токенов:
- **Access Token** (живёт короткое время, передаётся в заголовке `Authorization: Bearer <token>`)
- **Refresh Token** (используется для обновления Access токена)

**Пример заголовка:**
```http
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

---

## 🧠 Алгоритм FSRS

Мы используем **Free Spaced Repetition Scheduler** — современный алгоритм, который адаптируется под вашу память. В отличие от классических алгоритмов, FSRS учитывает:
- Сложность карточки
- Историю забываний
- Время, затраченное на ответ

Это позволяет сократить время обучения на 30-40% при сохранении того же уровня запоминания.
